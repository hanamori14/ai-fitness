# ai-fitness
https://adorable-flan-4d0f2b.netlify.app

商品の説明　：AIフィットネスアプリ - 月額プレミアムプラン（500円/月・税込）

スマートフォンのカメラでトレーニングフォームをリアルタイムチェック。
MediaPipe技術により、5種目（スクワット、プッシュアップ、アームカール、サイドレイズ、ショルダープレス）のフォーム確認を補助します。

【主な機能】
・リアルタイムフォーム確認補助
・音声による参考フィードバック
・トレーニング記録の自動保存
・カレンダー・グラフによる進捗管理

【プライバシー保護】
カメラ映像はスマートフォン内でのみ処理され、サーバーに送信されません。

【初回特典】7日間無料トライアル
【対象】13歳以上（未成年者は保護者の同意が必要）
